# Проект: Сайт кафе

Ссылка на проект: https://kafe-4877a1.webflow.io/

![Иллюстрация к проекту](https://github.com/e-tyatte/kafe/blob/main/Без%20имени.png)

## Общее описание

Проект представляет собой многофункциональный веб-сайт для современного кафе, сочетающего в себе онлайн-витрину продукции, блог с новостями и полноценный интернет-магазин кофе. Сайт разработан с акцентом на визуальную эстетику, удобство навигации и функциональность, позволяя посетителям не только знакомиться с ассортиментом, но и совершать покупки напрямую.

## Основной функционал

### 1. Демонстрация кофе и десертов  
- Витрина напитков и десертов с высококачественными изображениями.  

### 2. Новости (блог) кафе  
- Раздел с регулярными публикациями: анонсы мероприятий, новые меню, истории о бариста, кофейные лайфхаки.  
- Возможность подписки на обновления.

### 3. Магазин кофе  
- Каталог свежеобжаренного кофе в зёрнах, молотом виде и в капсулах.  
- Детальные карточки товаров с информацией об обжарке, регионе происхождения, вкусовых нотах.  
- Корзина и процесс оформления заказа (интеграция с платёжными системами через Webflow Ecommerce).  
- Возможность выбора помола и объёма упаковки.  

## Используемые технологии и сервисы

### Webflow ([https://webflow.com/](https://webflow.com/))  
- Использована AI-native платформа Webflow для визуальной разработки сайта без необходимости написания кода.  
- Применён встроенный **CMS** для удобного управления контентом новостей и каталога товаров.  
- Реализован **eCommerce-функционал** для продажи кофе: корзина, оформление заказа, управление инвентарём.  
- Адаптивная вёрстка «из коробки» — сайт корректно отображается на всех устройствах.  
- Встроенные SEO-инструменты и поддержка AEO (Answer Engine Optimization) для лучшей видимости в поиске.  
- Анимации и интерактивные элементы созданы с помощью визуального редактора Webflow.  

### Google Fonts  
- Подключены современные, читаемые и атмосферные шрифты с [Google Fonts](https://fonts.google.com/) для создания единой визуальной идентичности сайта.  
- Шрифтовая пара сочетает элегантность заголовков и практичность основного текста, подчёркивая кофейную эстетику бренда.  

## Целевая аудитория  
- Любители качественного кофе и десертов.  
- Постоянные клиенты кафе, интересующиеся новинками.  
- Онлайн-покупатели, желающие заказать кофе с доставкой домой или в офис.  
- Блогеры и гости, ищущие вдохновение в кофейной тематике.

## Преимущества решения  
- Быстрая и гибкая разработка благодаря Webflow: маркетинговая команда может самостоятельно обновлять контент.  
- Единая платформа для сайта, блога и магазина — упрощённое управление и аналитика.  
- Профессиональный внешний вид без привлечения разработчиков на постоянной основе.  
- Масштабируемость: в будущем можно легко добавить онлайн-бронирование столиков, лояльность, интеграцию с соцсетями и др.
